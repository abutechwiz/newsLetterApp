# NewsLetterApp




A newsLetter App which allows users to subscribe to a newsletter and receive emails.



The emails signed up using my app can be accessed in mailchimp.com/audiences and can be converted to csv format



# Technologies used:
   -> ExpressJS
   -> NodeJS
   -> MailChimp API
   -> Bootstrap
